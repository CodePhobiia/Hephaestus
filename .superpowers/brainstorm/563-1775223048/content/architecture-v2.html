Incorporating all 6 adjustments — UoW, decoupled events, content store, split projection, injectable IDs

Final Module Layout

src/hephaestus/forgebase/
  domain/ — pure models, zero I/O, zero imports
    models.py         # Vault, Source, Page, Claim, Link, Workbook...
    values.py         # EntityId types, Version, Revision (no generation)
    event_types.py    # Domain event schemas & taxonomy
    merge.py          # Merge rules, version reconciliation
    conflicts.py      # Conflict detection predicates
    enums.py          # ClaimStatus, PageType, JobStatus...
  repository/ — abstract contracts, imports only domain/
    vault_repo.py
    source_repo.py
    page_repo.py
    claim_repo.py
    link_repo.py
    workbook_repo.py
    job_repo.py
    content_store.py  # Abstract blob/artifact content contract
    uow.py            # UnitOfWork: repos + outbox + commit/rollback
  store/ — concrete implementations
    sqlite/            # aiosqlite UoW + repos + content
    postgres/          # asyncpg UoW + repos + content
    blobs/             # local FS / S3 content store impls
    migrations/        # Schema migrations (both backends)
  service/ — business logic, imports domain/ + repository/ only
    vault_service.py
    ingest_service.py
    branch_service.py
    merge_service.py
    compile_service.py
    lint_service.py
    id_generator.py   # Injectable ULID generation policy
  events/ — delivery infrastructure, wired by factory
    dispatcher.py      # Background outbox poller/deliverer
    consumers.py       # Durable consumer registry
    fanout.py          # Optional post-commit in-memory fanout
  query/ — read-model logic
    branch_view.py     # Branch-state read-through (COW overlay)
    diff.py            # Derived diff/review projection
  projection/ — external materialized projections
    git_mirror.py      # Git workspace projection (side-effecting)
  factory.py            # Sole composition root: config → wired ForgeBase

Final Dependency Rules

Layer May Import Key Change
domain/ nothing + merge.py, conflicts.py, event_types.py
repository/ domain/ + uow.py, content_store.py
store/ domain/, repository/ —
service/ domain/, repository/ NO direct events/ import
events/ domain/, repository/ Wired by factory, not by service
